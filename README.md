# Calorie Calculator Challenge

This is a small JavaScript project that calculates the **maximum daily calories** based on a person's **weight (kg) and age (years)**. I created it as part of my JS learning exercises.

## What the project does

- Takes user input for weight and age
- Calculates maximum recommended calories
- Displays the result dynamically in the browser

## Technologies

- HTML
- CSS
- JavaScript

## How to run

1. Open `challenge-01.html` in your browser
2. Enter your weight and age
3. See the calculated maximum daily calories
